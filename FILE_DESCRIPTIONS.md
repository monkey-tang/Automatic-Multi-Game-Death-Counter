# File and Folder Descriptions (Simple)

Simple descriptions for GitHub file explorer reference:

## Files

- **DeathCounterInstaller.exe** - Latest installer executable (v1.5) with Python version compatibility checking. Uses Git LFS.

- **README.md** - Documentation with installation instructions and troubleshooting.

- **.gitattributes** - Git LFS configuration for tracking large binaries.

- **.gitignore** - Git ignore rules for excluding temporary files and build artifacts.

- **cleanup_github_repo.ps1** - PowerShell script to organize repository structure.

- **create_release.ps1** - PowerShell script for creating GitHub releases.

## Folders

- **All Contents/** - Main application files extracted by the installer.

- **Debug.Test/** - Debug and testing utilities for troubleshooting.

- **Edit/** - Development scripts and source files for building the installer.
